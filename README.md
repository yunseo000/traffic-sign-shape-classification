# Traffic Sign Shape Classification

컴퓨터 비전 기초 기법을 이용해 교통 표지판의 기본 형태를 분류하는 팀 프로젝트입니다.

## 주제

이미지에 있는 표지판 모양을 보고 다음 3개 클래스로 분류합니다.

- `circle`: 원형 표지판
- `triangle`: 삼각형 표지판
- `square`: 사각형 표지판

실제 교통 표지판 인식은 색상, 글자, 아이콘, 촬영 각도 등 다양한 요소를 다루지만, 이 프로젝트에서는 난이도를 낮추기 위해 표지판의 큰 형태에 집중했습니다.

## 데이터

데이터는 `src/generate_dataset.py`로 생성한 합성 이미지 데이터입니다. 각 이미지는 64x64 PNG이며, 배경 노이즈, 위치 변화, 크기 변화, 회전, 밝기 변화를 넣어 단순 암기가 아니라 이미지 패턴을 학습하도록 구성했습니다.

생성 후 데이터는 `data/sign_shapes/` 아래에 저장됩니다.

```text
data/sign_shapes/
  circle/
  triangle/
  square/
```

## 사용 모델

복잡한 사전학습 모델 대신, 이미지 전처리와 특징 추출 과정을 명확하게 보여주기 위해 직접 구현한 작은 신경망 분류기를 사용했습니다.

처리 과정은 다음과 같습니다.

1. 이미지를 회색조로 변환
2. 픽셀 값을 0~1 범위로 정규화
3. 64x64 이미지를 32x32 특징으로 축소
4. 은닉층 1개를 가진 신경망으로 각 클래스 확률 예측
5. Accuracy와 confusion matrix로 평가

## 실행 방법

```bash
python src/generate_dataset.py
python src/train.py
python src/predict.py
```

결과 파일은 `outputs/` 폴더에 저장됩니다.

- `model.npz`: 학습된 모델 가중치
- `metrics.json`: 정확도와 클래스별 성능
- `confusion_matrix.png`: 혼동행렬 이미지
- `sample_predictions.png`: 예측 결과 예시 이미지

## 발표/PPT 구성 예시

1. 주제 선정 이유
   - 자율주행, 운전자 보조 시스템에서 교통 표지판 인식은 중요한 컴퓨터 비전 문제입니다.
   - 이 프로젝트는 그중 가장 기본 단계인 표지판 형태 분류를 실험합니다.

2. 데이터 소개
   - 원형, 삼각형, 사각형 총 3개 클래스
   - 합성 이미지 데이터 사용
   - 위치, 크기, 회전, 노이즈를 무작위로 적용

3. 전처리
   - 이미지 크기 64x64로 통일
   - 회색조 변환
   - 16x16 특징 벡터로 축소

4. 모델
   - 은닉층 1개를 가진 간단한 신경망 사용
   - 각 이미지가 세 클래스 중 어디에 속하는지 확률로 예측

5. 결과 해석
   - 정확도와 혼동행렬을 통해 모델이 어떤 형태를 잘 맞히는지 분석
   - 삼각형은 모서리 방향이 뚜렷해 잘 구분되는 편입니다.
   - 회전이 큰 사각형은 일부 이미지에서 원형과 헷갈릴 수 있습니다.

6. 한계와 개선점
   - 실제 사진 데이터가 아니라 합성 데이터이므로 현실 반영에는 한계가 있습니다.
   - 실제 교통 표지판 데이터셋을 추가하면 더 실용적인 모델로 확장할 수 있습니다.
   - CNN 또는 MobileNet 같은 딥러닝 모델로 성능 비교 실험을 할 수 있습니다.
